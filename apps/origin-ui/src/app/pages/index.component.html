Beautiful UI components built with Tailwind CSS and Zoneless Angular.

Origin UI is an extensive collection of copy-and-paste components for quickly building app UIs. It‘s free, open-source, and ready to drop into your projects.

This project is not affiliated with the original Origin UI.

Components

  • Input and Textarea
  • Button
  • Checkbox, Radio, and Switch
  • Select
  • Slider
  • Dialog
  • Tooltip
  • Dropdown and Popover
  • Avatar and Badge
  • Tabs
